## 📌 Sobre o projeto

Uma aplicação web simples e funcional que permite ao usuário
criar, visualizar e acompanhar suas rotinas diárias — ajudando
a manter o foco e a produtividade no dia a dia.

---

## ✨ Funcionalidades

- ➕ Adicionar tarefas à rotina diária
- ✅ Marcar tarefas como concluídas
- 🗂️ Visualizar a rotina completa do dia
- 🌐 Funciona direto no navegador, sem instalação

---

## 🚀 Como usar

```bash
# 1. Clone o repositório
git clone https://github.com/maltarodrigues98-cpu/gerador-de-rotina.git

# 2. Acesse a pasta
cd gerador-de-rotina

# 3. Abra o arquivo no navegador
Abra o arquivo index.html no seu navegador
```
